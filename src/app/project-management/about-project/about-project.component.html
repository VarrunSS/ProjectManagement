An Approach to Effective Project Management by Using Predictive Analytics

This is an application with which we could predict the right resource for the Project. Enter your project task summary and choose the project team and difficulty of the task to determine the best suitable resource for the project.

How it works?
  1. Historical data about the employees and their assignment are used as the base to develop the model
  2. Using the model, the data can be predicted with the help of Google Prediction API for the given input from user
  3. The output is displayed in form of charts to provide best user interface.
